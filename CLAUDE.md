# LongmontAI Project Instructions

**Deployed Site:** https://longmontai.com

## Visual Verification Workflow

When making visual/UI improvements to this site:

1. **Commit and push** changes when improvements are complete
2. **Wait 1-2 minutes** for deployment to propagate
3. **Verify with Puppeteer** - take a screenshot and check the changes visually
4. **If all looks good** - task is complete
5. **If there's a visual issue**:
   - Fix the issue
   - Commit and push again
   - Wait 1-2 minutes
   - Re-check with Puppeteer
   - Repeat up to **3 times maximum**
6. **After 3 failed attempts** - notify the user before proceeding further
